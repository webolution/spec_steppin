FedCM API Demo (Relying Party)

This demo simulates a website (Relying Party - RP) initiating a federated sign-in flow using the FedCM API. It will attempt to communicate with a hypothetical Identity Provider (IdP).

Important: FedCM is an experimental technology. You need a browser that supports it (e.g., Chrome, Edge with flags enabled, or Firefox Nightly). The `configURL` and `clientId` below are placeholders; a real IdP would provide these.

Click the "Sign in with FedCM" button to begin.
